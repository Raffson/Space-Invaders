Helper function for AR::SpaceInvaders::CollisionHandler that checks whether or not a shield got hit.

Parameters
bulletThe bullet which we're using to do this check.
shieldThe shield which we're using to do this check.
Returns
True if the bullet is close enough to the shield, otherwise false.

This function is needed since AR::inKillRange uses a radius which we don't want in case of a shield, therefore we use this adapted function.
